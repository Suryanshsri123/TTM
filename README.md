Task Tracking Manager (TTM)

Tech Stack:
Frontend: React (Vite)
Backend: Node.js, Express
Database: MongoDB (Atlas)
Deployment: Vercel (frontend), Railway (backend)

Features:
- User Authentication (JWT)
- Create, Read, Update, Delete Tasks
- Filter tasks by status (Todo / Done)
- Secure API with token

Live App:
https://your-vercel-url

Backend API:
https://ttm-production-3f32.up.railway.app

How to Run Locally:
1. Clone repo
2. Backend:
   cd backend
   npm install
   npm run dev
3. Frontend:
   cd frontend
   npm install
   npm run dev

Notes:
- Environment variables required for MongoDB and JWT secret
